# SassAndPuff

A collection of sass mixins for helping to design responsive websites.

## Installation

Add this line to your application's Gemfile:

    gem 'sass_and_puff'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sass_and_puff
